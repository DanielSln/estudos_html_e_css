Principais Formatações

Negrito ou Destaque

Nessa frase temos um termo em negrito usando a tag "b" (não semântica)

Já nessa frase, temos termo em destaque utilizando a tag "strong" (semântica)

Itálico ou Ênfase

Nesta frase, temos um termo em itálico usando a tag "i" (não semântica)

Entretanto, temos um termo em ênfase usando a tag "em" (semântica)

Texto Marcado

Podemos também criar um texto marcado "mark"

E no outro parágrafo temos outro texto marcado no final.

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Para criar um texto grande, é só utilizar a tag "big" (não semântica), além disso, para criar um texto pequeno utilize a tag "small"

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, utilizando a tag "del"

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag "ins" (semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+ 3 usando a tag "sup"

Texto Subscrito

Para inserir coisas do tipo H20 usando a tag "sub"
